🎮 자기소개 및 지원 동기

저는 업무를 수행할 때 '어떻게 구현할 것인가'보다 먼저 무엇을 만들어야 하며 왜 필요한지를 정의하는 것을 중요하게 생각합니다. 기능의 의도와 사용자 경험을 명확히 이해해야 구현 방향이 흔들리지 않고, 유지보수성과 확장성까지 고려한 개발이 가능하다고 믿기 때문입니다.


경일게임아카데미에서 게임 클라이언트 개발의 기초부터 심화까지 학습하며 개발자로서의 기반을 다졌고, 이후 지앤아이소프트 인턴으로 근무하며 실제 라이브 서비스 환경에서 버그 수정과 신규 기능 개발을 경험했습니다. 현재는 스노우파이프에서 정규직 클라이언트 프로그래머로 근무하며 AR 콘텐츠, GPS 기반 지도 시스템, 연출 제어 구조 등 다양한 실무 과제를 수행하며 기술적 깊이를 확장하고 있습니다.


저는 게임 클라이언트 개발이 단순히 화면을 구현하는 역할이 아니라, 유저 경험을 안정적으로 전달하기 위한 구조와 흐름을 설계하는 일이라고 생각합니다. 이러한 관점에서 쌓아온 경험을 바탕으로 더 완성도 높은 게임 경험을 제공하는 데 기여하고자 합니다.

⚡ 핵심 역량 및 성과

  • 시스템 설계 역량: 로비에서 다양한 연출이 동시에 실행되며 순서가 꼬이는 경합 문제가 반복적으로 발생하였습니다. 이를 해결하기 위해 우선순위 기반 연출 스케줄링 시스템(Scheduler + Director 구조)을 독립적으로 설계·구현하여, 연출 충돌 없이 순차 실행이 가능한 구조로 개선하였습니다.
  • 툴 제작 및 자동화: JSON 로우데이터 기반 미니게임 맵을 동적으로 구성하는 전용 에디터 툴을 제작하여 배치·시뮬레이션·저장·로드 기능을 통합함으로써 반복 작업을 줄이고 맵 제작 효율을 향상시켰습니다. 또한 Attribute 기반 테이블·리소스 검증 시스템을 구축하여 배포 전 데이터 오류를 자동 탐지하고, 검증된 ID 기반 드랍다운 UI를 제공해 QA 효율을 높였습니다.
  • AR 및 GPS 기반 콘텐츠 구현: AR 환경에서 3D 월드 좌표를 2D UI 좌표로 변환하는 시스템을 설계하여 탐색 판정 및 콘텐츠 구동을 안정화하였고, AR On/Off 전환을 고려한 공통 로직 구조로 에디터·모바일 환경 모두 대응하였습니다. 또한 GOMap과 OSM을 활용해 위도·경도 데이터를 월드 좌표로 변환하는 GPS 기반 지도 콘텐츠를 구현하였으며, RenderTexture 기반 분리 구조로 레이아웃 변화에도 안정적으로 표시되도록 개선하였습니다.

🤝 협업 및 문제 해결 경험

  • 구조적 문제 해결 경험: 저는 문제를 해결할 때 단순히 증상을 수정하기보다, 원인을 구조적으로 제거하여 재발 가능성을 줄이는 방식을 선호합니다. 로비 연출 경합 문제 역시 단순 조건 분기가 아닌 스케줄링 구조로 개선하여 동일한 문제가 반복되지 않도록 해결하였습니다.
  • 셰이더 협업 기반 그래픽 개선: AR 소환 콘텐츠 내 미니게임에서는 탑뷰 카메라 화면을 정면 카메라에 오버레이하는 구조로 인해, Perspective 환경임에도 정육면체의 앞·뒤 선에서 원근감이 충분히 표현되지 않는 문제가 있었습니다. 선마다 개별 머티리얼을 적용하는 방식이 논의되었으나, 맵툴에서 앞뒤를 구분하기 어렵고 대각선까지 고려하면 머티리얼이 과도하게 늘어나 유지보수 부담이 커질 수 있었습니다. 이에 저는 카메라 거리 기반으로 픽셀 단위 명암을 조절하는 방식을 제안하였고, 셰이더 담당자와 협업하여 적용 가능 여부를 검증했습니다. 결과적으로 머티리얼 증가 없이도 자연스러운 원근감이 표현되었고, 데이터 관리 부담 또한 줄일 수 있었습니다.
  • 라이브 서비스 대응 경험: 지앤아이소프트 인턴 시절, 실제 서비스 중인 '조선협객전 M'의 버그 수정과 신규 시스템 (PVP 흉신악살, 문파 기능 등)을 빠르게 도입하며 실 사용자 환경에서의 대응 능력과 코드 안정성의 중요성을 직접 체득하였습니다.

🚀 입사 후 포부

입사 후에는 빠르게 코드베이스와 업무 흐름을 파악하고, 기능 구현뿐 아니라 구조적 안정성과 유지보수성을 높이는 방향으로 기여하고자 합니다. 특히 라이브 서비스 환경에서 발생할 수 있는 연출 충돌, 데이터 오류, 플랫폼 이슈 등을 사전에 예방할 수 있는 구조와 검증 체계를 구축하는 데 강점을 발휘할 수 있습니다.


또한 에디터 툴 제작 및 데이터 검증 자동화 경험을 바탕으로 팀 전체의 생산성을 높이고, 반복 작업을 줄여 개발 효율을 개선하겠습니다. 최근에는 셰이더 및 그래픽 파이프라인에도 관심을 가지고 학습하고 있으며, 실제 프로젝트에서도 거리 기반 명암 처리 방식 제안을 통해 시각적 완성도를 개선한 경험이 있습니다. 앞으로도 시스템 설계와 그래픽 요소를 유기적으로 연결하여 완성도 높은 사용자 경험을 구현하는 개발자가 되겠습니다.


장기적으로는 폭넓은 게임 클라이언트 기술에 대한 깊은 이해를 갖추고, 팀의 기술적 방향을 함께 고민하고 이끌 수 있는 시니어 프로그래머로 성장하겠습니다. 좋은 게임을 만드는 데 있어 기술적 기반을 든든히 다듬는 역할을 맡고 싶습니다.
