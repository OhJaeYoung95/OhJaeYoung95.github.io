🎮 자기소개 및 지원 동기

저는 게임을 단순한 오락이 아니라, 기술과 감성이 결합된 인터랙티브 경험이라고 생각합니다. 그 경험을 직접 설계하고 구현하는 클라이언트 프로그래머라는 직무에 매력을 느껴 이 길을 선택했습니다.


경일게임아카데미에서 게임 클라이언트 개발의 기초부터 심화까지 체계적으로 학습하며 개발자로서의 기반을 다졌습니다. 이후 지앤아이소프트에서의 인턴 경험을 통해 실제 라이브 서비스 환경에서 빠른 버그 대응과 기능 개발을 경험하였고, 현재는 스노우파이프에서 정규직 클라이언트 프로그래머로서 AR 콘텐츠, GPS 기반 지도 시스템, 연출 스케줄링 등 다양한 기술적 도전을 수행하며 성장하고 있습니다.


앞으로도 플레이어가 더 몰입할 수 있는 게임 경험을 만드는 데 기여하고 싶으며, 기술적 깊이와 실무 감각을 겸비한 클라이언트 프로그래머로 계속 발전하고자 합니다.

⚡ 핵심 역량 및 성과

  • AR 콘텐츠 구현: 3D 월드 좌표를 2D UI 좌표로 변환하는 시스템을 직접 설계하여 AR 환경에서도 안정적인 탐색 판정 및 콘텐츠 구동을 가능하게 하였습니다. AR On/Off 전환을 고려한 공통 게임 로직 구조를 설계해 에디터·모바일 환경 모두 대응하였습니다.
  • 시스템 설계 역량: 로비 내 복수 연출의 동시 실행 경합 문제를 해결하기 위해 우선순위 기반 연출 스케줄링 시스템(Scheduler + Director 구조)을 독립적으로 설계·구현하였습니다.
  • 에디터 툴 제작: JSON 로우데이터 기반 미니게임 맵을 동적으로 구성하는 전용 에디터 툴을 제작하여 배치·시뮬레이션·저장·로드 기능을 통합함으로써 맵 제작 효율을 크게 향상시켰습니다.
  • 데이터 품질 자동화: Attribute 기반 테이블·리소스 검증 시스템을 구축하여 데이터 오류를 배포 전에 자동 탐지하고, 검증된 ID 기반 드랍다운 UI를 제공해 QA 효율을 높였습니다.
  • GPS/지도 기술: GOMap과 OSM을 활용해 위도·경도 데이터를 월드 좌표로 변환하는 GPS 기반 지도 콘텐츠를 구현하였으며, RenderTexture 기반 분리 구조로 레이아웃 변화에도 안정적으로 표시되도록 개선하였습니다.

🤝 협업 및 문제 해결 경험

  • 로비 연출 경합 문제 해결: 일일 집계, 성장 연출 등 여러 연출이 동시에 트리거되어 순서가 꼬이는 경합 이슈가 반복적으로 발생하였습니다. 이를 해결하기 위해 우선순위 기반 연출 스케줄링 시스템(Scheduler + Director)을 독립적으로 설계하여 연출 충돌 없이 순차·우선 실행이 가능한 구조를 구축하였습니다.
  • 전투 시스템 레벨업 버그 수정: 서버 판정 기반 전투에서 레벨업이 전투 도중 발생할 경우 경험치 연출 흐름이 어긋나는 버그가 있었습니다. 전투 시작과 종료 데이터를 분리하는 구조로 리팩토링하여 레벨업 반영 시점을 명확히 하고, 연출 흐름을 안정적으로 개선하였습니다.
  • 라이브 서비스 대응 경험: 지앤아이소프트 인턴 시절, 실제 서비스 중인 '조선협객전 M'의 버그 수정과 신규 시스템(PVP 흉신악살, 문파 기능 등)을 빠르게 도입하며 실 사용자 환경에서의 대응 능력과 코드 안정성의 중요성을 직접 체득하였습니다.
  • 에디터 툴을 통한 팀 협업 효율화: 맵 에디터 툴 제작 시 기획자와 충분히 소통하여 요구 사항을 기능 단위로 명확히 정의하고, 시뮬레이션 기능을 추가해 실제 인게임과 동일한 환경에서 검증할 수 있도록 하였습니다. 이를 통해 기획·개발 간 커뮤니케이션 비용을 줄이고 반복 수정 사이클을 단축하였습니다.

🚀 입사 후 포부

입사 후에는 기존 업무에 빠르게 적응하고, 클라이언트 개발에서 실질적인 기여를 만들어 나가겠습니다. AR, GPS, 실시간 시스템 설계 등 현재 쌓아온 경험을 바탕으로, 팀이 필요로 하는 곳에 주도적으로 참여하고 완성도 높은 결과물을 함께 만들어 가고 싶습니다.


또한 에디터 툴 제작이나 검증 자동화 시스템 등 팀의 개발 효율을 높이는 방향으로도 기여하겠습니다. 혼자만 잘하는 프로그래머가 아니라, 팀 전체의 생산성과 코드 품질을 함께 높여가는 동료가 되는 것이 목표입니다.


장기적으로는 폭넓은 게임 클라이언트 기술에 대한 깊은 이해를 갖추고, 팀의 기술적 방향을 함께 고민하고 이끌 수 있는 시니어 프로그래머로 성장하겠습니다. 좋은 게임을 만드는 데 있어 기술적 기반을 든든히 다듬는 역할을 맡고 싶습니다.
